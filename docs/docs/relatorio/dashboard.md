# Dashboards de relatório

Um **Dashboard** é um painel visual composto por relatórios adicionados como widgets.

:::warning Importante
Você precisa criar relatórios antes de utilizá-los em dashboards.
:::

![Tela de dashboards de relatório](assets/image-2.png)

## Criando um Novo Dashboard

Clique em **Novo dashboard**.

Preencha:

### Nome (Obrigatório)
Exemplos:
- Visão Comercial
- Indicadores Semanais
- Performance Atendimento

### Descrição (Opcional)

Uma breve descrição do que o relatório informa.

### Compartilhamento
- Privado
- Empresa inteira
- Cargos específicos

Clique em **Salvar dashboard**.

## Filtro Global de Datas

Todo dashboard possui um filtro global de período.

Esse filtro:

- Afeta todos os widgets simultaneamente
- Permite definir data inicial e final
- Oferece atalhos como “Últimos 30 dias”

## Adicionando Widgets

Dentro do dashboard, clique em **Adicionar widget**.

![Modal de adicionar widget](assets/image-4.png)

#### Relatório
Selecione um relatório já criado.

![Configuração de widget de relatório](assets/image-5.png)

#### Título (Opcional)
Permite personalizar o nome exibido no dashboard.

#### Tipo de Exibição
Escolha o formato visual:

- Barras
- Linha
- Área

Clique em **Adicionar**.

O relatório agora será exibido como widget no dashboard.

## Boas Práticas

- Utilize nomes claros nos relatórios
- Evite períodos excessivamente longos
- Crie relatórios específicos para cada necessidade
- Use dashboards para visão gerencial
- Use relatórios individuais para análises detalhadas
