# Versão 1.33.0

_Publicada em 14/07/2026._

## 🔄 Demonstrações sempre atuais

- Ambientes de demonstração preparados pela FalaMais agora mantêm tarefas,
  compromissos, negociações e conversas em datas próximas ao dia da
  apresentação.
- Prazos relativos continuam fazendo sentido com o passar dos dias. Uma tarefa
  preparada para “amanhã”, por exemplo, permanece como uma tarefa de amanhã na
  próxima apresentação.
- Indicadores, metas, propostas, campanhas de exemplo e dados de aquisição
  acompanham o mesmo período, deixando a história apresentada consistente em
  todas as áreas do sistema.
- A renovação acontece somente nos ambientes de demonstração habilitados para
  isso e não altera dados de contas de clientes.
