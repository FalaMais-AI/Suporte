# Versão 1.2.4

_Publicada em 21/05/2026._

## 🔧 Melhorias

- **Formulários incorporados mais seguros contra configurações visuais inválidas.**
  O Conversion Hub agora valida melhor as cores e estilos personalizados usados
  nos formulários incorporados, reduzindo o risco de uma configuração visual
  quebrar a página onde o formulário aparece.

📖 **Saiba mais:** [Configuração de integrações](../configuracao/integracao.md) · [Contatos](../contatos.md)

## 🐛 Correções

- **Envio de emails com tipos mais consistentes.** Ajustamos a dependência de
  tipos usada no envio de emails para acompanhar a versão real do provedor SMTP
  utilizado pelo sistema.
