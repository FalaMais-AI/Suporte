# Versão 1.20.8

_Publicada em 06/07/2026._

## 🐛 Correções

- Corrigimos um caso em que mensagens novas podiam chegar no WhatsApp do
  celular, mas não aparecer imediatamente na lista principal do FalaMais quando
  a conversa estava arquivada depois de uma migração ou troca de canal.
- A partir desta versão, quando o cliente responde de verdade, a conversa volta
  automaticamente para a lista ativa ou para a fila, mantendo o histórico no
  canal correto.
- Importações de histórico e sincronizações antigas continuam sem reabrir
  conversas arquivadas, evitando que conversas antigas voltem para a operação
  sem uma nova resposta do cliente.
