# Versão 1.31.2

_Publicada em 10/07/2026._

## 🔧 Melhorias

- **Acompanhamento claro das mídias:** enquanto um áudio, imagem, vídeo ou
  documento ainda está sendo preparado, a conversa mostra esse processamento.
  Se o arquivo original já não estiver disponível, o atendente recebe uma
  orientação clara para solicitar o reenvio.
- **Recuperação automática:** falhas temporárias no recebimento de arquivos
  passam por novas tentativas controladas, reduzindo mensagens que ficam apenas
  como um marcador no histórico.

📖 **Saiba mais:** [Mensagens](../msg.md) ·
[Canais de atendimento](../configuracao/canais.md)

## 🐛 Correções

- **Mídias do WhatsApp Oficial:** áudios, imagens, vídeos e documentos recebidos
  voltam a carregar corretamente no atendimento web.
- **Envio para integrações:** eventos configurados para sistemas externos voltam
  a ser entregues normalmente, inclusive os que precisam de conexão segura.
- **Configurações respeitadas:** opções administrativas marcadas como
  desativadas deixam de ser ligadas por engano durante a inicialização.
