# Versão 1.22.1

_Publicada em 07/07/2026._

## 🐛 Correções

- Ao **mesclar contatos duplicados**, as ligações registradas agora acompanham
  o contato principal, e os telefones e e-mails adicionais não são mais
  duplicados quando coincidem com os dados principais.
- As **metas de vendas** só aceitam vendedores que realmente fazem parte da
  sua empresa.
- O **envio do áudio de uma ligação** ficou mais seguro: se algo falhar no
  meio do envio, nenhum arquivo fica perdido no armazenamento.
- Os **alertas gerenciais por e-mail** que falharem no envio são reenviados
  automaticamente no próximo ciclo, em vez de se perderem.
