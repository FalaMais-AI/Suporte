# Versão 1.17.0

_Publicada em 22/06/2026._

## ✨ Novidades

- Estreia a página de inscrição de eventos do FalaMais.AI, começando pelo
  **Mais Talks**. Quem se inscreve informa nome, e-mail, telefone, empresa e
  cargo — e ainda pode adicionar mais um convidado na mesma inscrição. Se o
  e-mail já tiver sido cadastrado para o evento, a página avisa na hora.
- As inscrições chegam automaticamente organizadas como contatos, prontas para
  o acompanhamento da equipe, sem planilhas ou digitação manual.
