# Versão 1.4.15

_Publicada em 01/06/2026._

## 🔧 Melhorias

- **Escolha o que acontece quando o paciente cancela.** Nas regras da Admed,
  agora você pode definir se a resposta negativa deve apenas deixar a consulta
  como não confirmada ou se o sistema deve realmente desmarcar o agendamento
  para liberar a vaga.
- **Liga/desliga mais claro nas automações.** A ativação das regras da Admed
  agora usa um switch separado da ação de atualizar prévia, deixando mais
  evidente quando a automação está ativa ou inativa.
