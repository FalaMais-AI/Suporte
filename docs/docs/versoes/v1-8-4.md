# Versão 1.8.4

_Publicada em 05/06/2026._

## Correções

- Corrigimos o envio de mensagens para contatos internacionais quando o número já vinha com código de país diferente do Brasil.
- O sistema agora preserva números internacionais válidos, como contatos com DDI `+1`, em vez de tentar adaptar automaticamente para o formato brasileiro.
- A validação de telefone continua avisando quando um número brasileiro está sem o DDI `55`, mas não bloqueia mais contatos internacionais válidos por engano.
