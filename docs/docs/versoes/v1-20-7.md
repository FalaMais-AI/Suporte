# Versão 1.20.7

_Publicada em 06/07/2026._

## 🐛 Correções

- A remoção de um canal do WhatsApp com transferência de histórico ficou mais
  estável para canais com muitas conversas.
- Quando a equipe transfere conversas de um canal para outro, o FalaMais passa
  a tratar melhor as combinações entre QR Code e API Oficial: QR para QR, API
  para API, API para QR e QR para API.
- Também reforçamos a preservação do histórico operacional quando duas conversas
  do mesmo contato precisam ser unificadas durante a transferência, mantendo
  mensagens, atendimento, memória da IA, roteamento e agendamentos vinculados ao
  canal ativo.
