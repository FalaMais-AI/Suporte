# Versão 1.3.1

_Publicada em 26/05/2026._

## ✨ Novidades

- **Envio de botões pela API pública.** Integrações externas agora podem enviar
  mensagens do WhatsApp com botões pela API pública em canais UAZAPI. O envio
  aceita botões de resposta rápida, abertura de link, cópia de código e ligação.
- **Exemplos mais claros na documentação da API.** A documentação interativa da
  API pública agora mostra exemplos separados para mensagem de texto, mensagem
  com botões e envio de mídia.

📖 **Saiba mais:** [Mensagens](../msg.md)

## 🔧 Melhorias

- **Controle de prévia de links pela API.** Integrações externas podem informar
  quando querem ativar ou desativar a prévia de links em mensagens de texto.

📖 **Saiba mais:** [Mensagens](../msg.md)
