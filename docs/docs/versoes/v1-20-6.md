# Versão 1.20.6

_Publicada em 06/07/2026._

## 🔧 Melhorias

- A IA ficou mais precisa ao decidir quando deve passar uma conversa para
  atendimento humano. Agora, mencionar equipe, consultor ou atendimento no texto
  do prompt não basta para pausar a conversa: a ação precisa estar configurada
  explicitamente pelo menu `@`.
- Informações simples que já estão no prompt, como endereço, localização,
  unidades e orientações de atendimento, continuam disponíveis para a IA
  responder normalmente.

📖 **Saiba mais:** [Prompt do assistente](../assistente/prompt.md) · [Autonomia da IA](../assistente/autonomia.md)

## 🐛 Correções

- Quando a passagem para humano é realmente acionada, a IA consegue enviar a
  mensagem final de transição ao cliente antes de deixar a conversa pausada para
  a equipe continuar.

📖 **Saiba mais:** [Mensagens](../msg.md) · [Prompt do assistente](../assistente/prompt.md)
