# Versão 1.20.1

_Publicada em 03/07/2026._

## 🐛 Correções

- A aba **Disparos** agora respeita corretamente a liberação específica de
  campanhas. Empresas com WhatsApp ativo, mas sem Disparos contratado ou
  habilitado, deixam de ver e acessar essa área.
- Corrigimos um cenário em que a tela podia ficar recarregando repetidamente
  após detectar uma nova versão do sistema. Agora, se o navegador continuar
  preso em uma versão antiga em cache, o FalaMais evita repetir a atualização em
  loop.
