# Versão 1.5.0

_Publicada em 02/06/2026._

## ✨ Novidades

- **Videochamada direto pelo atendimento.** Agora o operador pode criar uma
  videochamada na conversa com um botão rápido, revisar a mensagem antes de
  enviar e mandar o convite com botão de acesso quando o canal permitir.

- **Agendamento de videochamadas.** O convite também pode ser agendado para um
  horário futuro com mensagem personalizada. O link da reunião fica preservado
  no agendamento e, se o canal não permitir botão no disparo, o cliente recebe
  o link no texto da mensagem.
