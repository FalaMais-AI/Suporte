# Versão 1.3.4

_Publicada em 27/05/2026._

## 🔧 Melhorias

- **Erro mais claro ao criar templates.** Quando a Meta recusar um parâmetro,
  o sistema passa a mostrar mais detalhes da resposta original sempre que a Meta
  enviar essa informação.

📖 **Saiba mais:** [Mensagens](../msg.md) · [Canais — WhatsApp](../configuracao/canais.md)

## 🐛 Correções

- **Botões com link direto para WhatsApp.** O criador de templates agora avisa
  antes do envio quando o botão usa links como `wa.me` ou domínios do WhatsApp,
  que a Meta pode recusar em templates. Use uma página externa ou botão de
  resposta rápida nesses casos.

📖 **Saiba mais:** [Mensagens](../msg.md) · [Canais — WhatsApp](../configuracao/canais.md)
