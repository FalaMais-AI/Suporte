# Versão 1.8.7

_Publicada em 08/06/2026._

## 🐛 Correções

- Campanhas de disparo com intervalos longos entre mensagens ficaram mais equilibradas: uma campanha lenta não prende a fila e outras campanhas continuam avançando.
- Públicos que ainda estão vinculados a campanhas ativas agora ficam protegidos contra remoção acidental, evitando campanhas sem audiência.
- O acompanhamento de entrega dos disparos ficou mais confiável: falhas ao registrar eventos importantes agora são tratadas como erro visível.
- As métricas do WhatsApp Oficial validam melhor o período consultado, evitando filtros inválidos ou ambíguos.
- A passagem para atendimento humano e as ações de tags feitas pelo Copiloto ficaram mais claras quando algum registro não é salvo corretamente.
- Ajustes internos de leitura numérica reduzem interpretações ambíguas em eventos recebidos de integrações oficiais.
