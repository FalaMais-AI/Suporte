# Versão 1.4.8

_Publicada em 29/05/2026._

## 🐛 Correções

- **Retornos da Admed com diagnóstico mais claro.** Quando uma confirmação de
  paciente chega para um disparo que já não pode ser processado automaticamente,
  o sistema agora identifica melhor o motivo: disparo inexistente, unidade sem
  WhatsApp vinculado ou canal removido. Isso evita novas tentativas inúteis e
  facilita a correção pela operação.
