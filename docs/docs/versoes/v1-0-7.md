# Versão 1.0.7

_Publicada em 19/05/2026._

## 🐛 Correções

- As mensagens enviadas por conexões WhatsApp UAZAPI voltam a atualizar as
  confirmações de entrega e leitura corretamente. Isso corrige casos em que a
  mensagem já tinha sido entregue ou lida no WhatsApp, mas continuava aparecendo
  no FalaMais.AI apenas como enviada.

- A correção vale para todas as conexões UAZAPI atuais e para as próximas
  conexões criadas no sistema.
