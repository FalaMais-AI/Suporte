# Versão 1.34.0

_Publicada em 14/07/2026._

## ✨ Convites de acesso por email

- Ao adicionar uma pessoa à empresa, agora você pode escolher entre **enviar um
  convite por email** ou **definir uma senha inicial na hora**.
- No convite por email, a pessoa recebe um link para confirmar o endereço e
  escolher a própria senha antes do primeiro acesso.
- A lista de usuários mostra quando um convite está pendente ou expirado e
  permite enviar um novo convite quando necessário.
- O Super Admin também conta com as duas formas de criação ao adicionar um
  usuário a uma empresa.

📖 **Saiba mais:** [Convidar e gerenciar usuários](../configuracao/usuarios/usuarios.md)

## 🔒 Ativação mais segura

- O acesso enviado por email só é liberado depois que a própria pessoa abre o
  convite e define a senha.
- Cada link pode ser usado uma única vez e fica disponível por até 72 horas.
  Ao reenviar o convite, o link anterior deixa de funcionar.

📖 **Saiba mais:** [Convidar e gerenciar usuários](../configuracao/usuarios/usuarios.md)
