# Versão 1.0.2

_Publicada em 18/05/2026._

## 🐛 Correções

- As datas inseridas nos modelos de proposta agora aparecem em formato
  brasileiro, como `25/05/2026`, em vez de mostrar o valor técnico completo.
- A lista de campos exibida ao digitar `@` no editor de modelos voltou a rolar
  corretamente, facilitando encontrar e inserir funcionalidades/campos mais
  abaixo na lista.
