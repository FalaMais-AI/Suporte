# Versão 1.37.5

_Publicada em 21/07/2026._

## 🐛 Correções

- Campos criados para um modelo específico de negociação não aparecem mais
  automaticamente em todas as negociações. Eles ficam disponíveis somente
  quando o modelo de campos correspondente é aplicado.
- Ao ativar um layout próprio ou salvar um novo modelo a partir do layout
  padrão, o sistema preserva a separação dos campos exclusivos de outros
  modelos.

📖 **Saiba mais:** [Campos personalizados e modelos de layout](../configuracao/campos.md) e
[propostas e contratos](../funil/propostas.md).
