# Versão 1.8.2

_Publicada em 05/06/2026._

## 🔧 Melhorias

- A página de planos ficou mais direta para comparar os níveis **Essencial**, **Growth** e **Scale**, com uma leitura mais clara do que entra em cada etapa da operação.
- Removemos a calculadora dessa página para deixar a decisão mais objetiva e focada em estrutura, canais, automações e recursos de IA.
- Também reorganizamos os blocos da página para destacar melhor como a plataforma combina atendimento, CRM e automações no mesmo fluxo.
