# Versão 1.4.9

_Publicada em 29/05/2026._

## 🔧 Melhorias

- **Mais estabilidade em mensagens rápidas com mídia.** O sistema passou a
  tratar texto, mídia e botões de respostas rápidas como uma sequência única,
  deixando substituições e remoções de anexos mais previsíveis.
- **Documentação pública da API mais consistente.** A referência da API pública
  foi alinhada para reduzir diferenças entre o que é documentado e o que o
  sistema aceita.
- **Mais segurança em chaves e registros internos.** A plataforma recebeu
  validações mais rígidas para configurações sensíveis e registros operacionais
  mais seguros.
- **Mensagens agendadas com operação mais flexível.** O processamento de
  mensagens agendadas agora pode ser ajustado pela operação conforme o volume
  de envios, sem exigir uma nova alteração de código.

📖 **Saiba mais:** [Mensagens](../msg.md) · [Agenda](../agenda.md) · [Integrações](../configuracao/integracao.md) · [Segurança](../configuracao/seguranca.md)

## 🐛 Correções

- **Respostas com botões pela API pública preservam melhor o contexto.** Ao
  enviar uma mensagem com botões respondendo a uma mensagem anterior, o contexto
  da resposta agora segue o caminho correto e fica registrado na conversa.
- **Filtros de agenda mais consistentes.** A listagem de tarefas e mensagens
  agendadas passou a validar os status aceitos de forma centralizada, reduzindo
  divergências em filtros.
- **Integração Admed com rotas mais diretas.** Consultas de regras, unidades,
  modelos e histórico usam os caminhos específicos de cada informação,
  evitando dependência de uma visão agregada.
