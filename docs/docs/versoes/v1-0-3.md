# Versão 1.0.3

_Publicada em 18/05/2026._

## 🐛 Correções

- A busca da lista de mensagens e da fila agora encontra conversas pelo nome
  do lead ou pelo telefone, mesmo quando existem acentos, apóstrofos, espaços,
  parênteses, hífens ou `+` em formatos diferentes.
- A lista de conversas deixou de trazer resultados só porque a palavra buscada
  apareceu dentro de uma mensagem antiga. Para procurar no conteúdo da conversa,
  continue usando a busca dentro do chat aberto.
