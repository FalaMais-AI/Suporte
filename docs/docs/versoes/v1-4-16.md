# Versão 1.4.16

_Publicada em 01/06/2026._

## 🐛 Correções

- **Troca de conta e acesso ficaram mais consistentes.** Quando uma pessoa é
  adicionada a uma empresa ativa, ela consegue entrar normalmente mesmo que
  tenha histórico antigo em uma empresa inativa. Empresas inativas deixam de
  aparecer na troca de conta, enquanto contas com período de teste encerrado
  continuam visíveis para que o pagamento possa ser regularizado.
