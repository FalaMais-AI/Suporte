# Versão 1.17.11

_Publicada em 26/06/2026._

## 🐛 Correções

- O editor de prompt agora respeita o modo escolhido. Clicar no modo visual não
  muda mais automaticamente para o modo raw, então você pode revisar e editar o
  prompt no formato que preferir.
- O modo raw também permite buscar funções com `@`, incluindo ações da IA,
  equipes, usuários, funis, tags, fluxos, listas e integrações personalizadas.
  Ao selecionar uma opção, o sistema insere a referência correta no prompt.
