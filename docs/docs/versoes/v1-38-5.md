# Versão 1.38.5

_Publicada em 24/07/2026._

## 🐛 Correções

- Corrigimos o editor de prompt para que o cursor, a seleção e a exclusão de
  texto acompanhem exatamente o que aparece na tela, inclusive em textos
  longos.
- As referências de ações, dados e mídias continuam fáceis de conferir pelo
  nome durante a edição, sem caracteres invisíveis ou desalinhamento do texto.
- Você também pode escrever uma referência manualmente sem o editor completar
  o colchete antes de terminar o nome.
