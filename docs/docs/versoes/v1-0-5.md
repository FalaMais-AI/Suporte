# Versão 1.0.5

_Publicada em 18/05/2026._

## 🔧 Melhorias

- A busca da lista de Mensagens e da Fila ficou mais previsível: ao digitar um
  nome ou telefone, a lista só mostra conversas cujo lead corresponde a esse
  nome ou número, mesmo com acentos, apóstrofos, espaços, parênteses, hífens ou
  `+` no telefone.
- Enquanto a busca nova está carregando, a lista mostra o estado de
  carregamento em vez de manter conversas antigas na tela, evitando confusão
  visual.

📖 **Saiba mais:** [Mensagens](../msg.md)

## 🐛 Correções

- Termos que aparecem apenas no texto de mensagens antigas não trazem mais a
  conversa para a lista ou para a fila. A busca por texto dentro de uma conversa
  aberta continua funcionando normalmente.
- A aba de Grupos agora busca pelo nome do grupo de forma normalizada, sem usar
  o conteúdo das mensagens do grupo como critério.
