# Versão 1.10.5

_Publicada em 12/06/2026._

## 🔧 Melhorias

- Ao criar ou editar uma integração personalizada da IA, agora a tela pergunta claramente se a integração precisa de confirmação antes de executar. Para consultas simples, como ver resultados, status ou disponibilidade, é possível escolher **Não, executar direto**. Para ações sensíveis, como criar, alterar, enviar ou cobrar algo, mantenha **Sim, pedir confirmação**.
