# Versão 1.20.3

_Publicada em 03/07/2026._

## 🐛 Correções

- Quando alguém assume ou transfere uma conversa no atendimento, a IA passa a
  ficar pausada imediatamente e deixa de responder ao cliente nessa conversa.
- A proteção também vale se a conversa ainda aparecia como ativa por algum
  estado antigo ou se quem assumiu é um usuário administrador.
- A lista de conversas e os filtros de IA ativa/pausada passam a tratar toda
  conversa atribuída a uma pessoa como atendimento humano.
