# Versão 1.15.10

_Publicada em 18/06/2026._

## 🐛 Correções

- Os disparos agora continuam avançando quando apenas alguns contatos apresentam erro de envio. Esses contatos ficam marcados com falha, mas os próximos destinatários seguem na fila normalmente.
- Se o problema for de pagamento ou cobrança da conta, a campanha é pausada e mostra o motivo para que a equipe possa regularizar antes de retomar.
