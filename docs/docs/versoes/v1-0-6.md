# Versão 1.0.6

_Publicada em 19/05/2026._

## 🔧 Melhorias

- A configuração do painel ficou mais clara para ambientes de implantação,
  especialmente nas áreas de API, pagamentos, conexão em tempo real e liberação
  gradual de recursos. Isso reduz o risco de uma tela importante ficar
  incompleta por falta de configuração no ambiente.

📖 **Saiba mais:** [Pagamentos](../configuracao/pagamentos.md)

## 🐛 Correções

- A rotina de conferência de status das mensagens UAZAPI agora separa melhor os
  casos em que faltam credenciais dos erros reais do provedor. Isso deixa o
  diagnóstico de mensagens enviadas mais limpo para a operação.

📖 **Saiba mais:** [Mensagens](../msg.md) · [Canais — WhatsApp](../configuracao/canais.md)
