# Versão 1.10.0

_Publicada em 10/06/2026._

## ✨ Novidades

- A área de IA agora permite criar integrações HTTP personalizadas para conectar o agente a sistemas externos da empresa.
- Cada integração pode ter método, endpoint, autenticação, campos do JSON, status ativo/inativo e teste antes de usar em atendimento.
- Integrações ativas aparecem no editor de prompt pelo menu `@`, para que cada prompt escolha exatamente quais ferramentas a IA pode usar.

📖 **Saiba mais:** [IA e integrações personalizadas](../assistente/integracoes-personalizadas.md)

## 🔒 Segurança e controle

- Tokens e chaves de integração ficam protegidos e não reaparecem em texto claro depois de salvos.
- Por padrão, a IA precisa pedir confirmação do cliente na conversa antes de executar uma integração personalizada.
- As execuções respeitam empresa, prompt, status da integração e nível de autonomia configurado.

📖 **Saiba mais:** [IA e integrações personalizadas](../assistente/integracoes-personalizadas.md)
