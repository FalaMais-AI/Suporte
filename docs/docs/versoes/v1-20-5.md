# Versão 1.20.5

_Publicada em 03/07/2026._

## ✨ Novidades

- As regras de follow-up agora têm a opção **Enviar durante atendimento humano**.
  Quando ativada, a regra pode continuar acompanhando uma conversa mesmo se ela
  estiver atribuída a um atendente ou com a IA pausada.
- Para regras de **sem resposta**, o tempo passa a contar a partir da última
  mensagem enviada pela IA ou pelo atendente. Se o atendente enviar uma nova
  mensagem, o follow-up pendente é reagendado a partir desse novo envio.

📖 **Saiba mais:** [Regras de automação](../automacao/regras.md) · [Mensagens](../msg.md)

## 🔧 Melhorias

- A resposta do cliente continua tendo prioridade: se o cliente responder antes
  do disparo, o follow-up pendente é cancelado normalmente.
- Regras antigas continuam seguras por padrão. A nova opção nasce desligada e
  só muda o comportamento quando alguém ativa explicitamente na regra.

📖 **Saiba mais:** [Regras de automação](../automacao/regras.md)
