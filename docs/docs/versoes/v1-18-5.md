# Versão 1.18.5

_Publicada em 01/07/2026._

## 🐛 Correções

- Corrigimos o indicador **Atendimentos por demanda** do dashboard. Agora ele
  considera as conversas iniciadas no período selecionado que já foram
  encerradas, deixando o total e a lista de motivos alinhados com a exportação
  de relatórios.
- O mini-gráfico desse indicador também passa a seguir o mesmo critério do
  total, evitando diferenças quando atendimentos de um período são encerrados
  depois.
