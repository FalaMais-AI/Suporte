# Versão 1.10.4

_Publicada em 12/06/2026._

## 🐛 Correções

- Fluxos de automação ficaram mais estáveis em cenários em que uma ação do próprio fluxo envia uma mensagem. Isso evita reentradas repetidas e reduz o risco de uma automação responder em loop.
- Integrações personalizadas da IA agora conseguem responder melhor quando uma API retorna listas ou tabelas. Quando a integração traz vários itens, como jogos, cotações, pedidos ou registros, a IA recebe dados mais úteis para encontrar o item solicitado e responder com o resultado correto.
