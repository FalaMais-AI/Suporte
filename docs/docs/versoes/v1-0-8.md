# Versão 1.0.8

_Publicada em 19/05/2026._

## 🐛 Correções

- Melhoramos novamente a atualização dos risquinhos de mensagens enviadas pelo
  WhatsApp UAZAPI. Algumas conexões recebiam a confirmação do WhatsApp em outro
  formato e, por isso, mensagens entregues ou lidas podiam continuar aparecendo
  apenas como enviadas no FalaMais.AI.

- A correção vale para todas as conexões UAZAPI atuais e futuras.
