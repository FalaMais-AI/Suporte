Esta é a primeira versão registrada do FalaMais.AI. A partir de agora, toda
correção ou novidade da plataforma será publicada aqui, nesta página de
**Versões**, para você acompanhar o que muda a cada atualização.

  Inteligência Artificial.
- Criação desta seção de **Versões**, onde você acompanha tudo o que é
  adicionado, melhorado ou corrigido na plataforma.
