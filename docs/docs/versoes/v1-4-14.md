# Versão 1.4.14

_Publicada em 01/06/2026._

## 🐛 Correções

- **Mais clareza quando a proposta ainda está em rascunho.** Ao abrir um link
  público de proposta que ainda não foi enviada, o cliente agora vê um aviso
  explicando que o documento está disponível apenas para leitura naquele
  momento.
- **Assinatura bloqueada até o envio oficial.** Enquanto a proposta estiver em
  rascunho, os botões de aceite e assinatura ficam indisponíveis. Isso evita a
  sensação de erro quando o documento ainda não foi liberado pelo responsável.
- **Menos risco de compartilhar link antes da hora.** No painel interno, links
  públicos de propostas em rascunho não podem mais ser copiados ou abertos como
  se já estivessem prontos para envio.
