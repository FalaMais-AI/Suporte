# Versão 1.34.4

_Publicada em 15/07/2026._

## 🐛 Mensagens recebidas com mais segurança

- Corrigimos um caso em que formatos incomuns enviados pelo provedor do
  WhatsApp podiam impedir a entrada de uma mensagem válida no histórico.
- A conversa agora continua recebendo normalmente mesmo quando informações
  complementares da mensagem chegam em uma estrutura inesperada.
- O ajuste também protege recuperações de histórico, reduzindo o risco de uma
  única mensagem incompatível interromper as demais.
