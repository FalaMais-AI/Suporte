# Versão 1.0.4

_Publicada em 18/05/2026._

## 🐛 Correções

- As mensagens enviadas por conexões UAZAPI voltam a mostrar corretamente o
  avanço de entrega e leitura no atendimento, incluindo os indicadores de dois
  riscos cinza e dois riscos azuis quando o WhatsApp confirma esses estados.
- Mensagens enviadas recentemente por UAZAPI que ficaram presas como apenas
  "enviadas" podem ser reconciliadas com o status real informado pelo
  provedor.
