# Versão 1.18.1

_Publicada em 29/06/2026._

## 🐛 Correções

- Notas enviadas por integrações na criação ou atualização de contatos voltam
  a aparecer também na aba **Notas** do contato. Assim, a equipe consegue ver o
  histórico importado no lugar esperado dentro do CRM.
- Quando a mesma integração reenviar exatamente a mesma nota, o sistema evita
  criar registros duplicados na aba **Notas**.
