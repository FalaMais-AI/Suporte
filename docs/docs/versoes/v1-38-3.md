# Versão 1.38.3

_Publicada em 23/07/2026._

## 🔧 Melhorias

- Os atendimentos, resumos, análises e acompanhamentos agora usam um modelo
  mais rápido e econômico como primeira opção.
- Conversas mais complexas podem avançar automaticamente para um modelo com
  maior capacidade de raciocínio, especialmente quando há várias ações ou
  informações ambíguas.
- O copiloto e a criação inicial do assistente passam a priorizar o modelo de
  maior qualidade, mantendo alternativas automáticas caso algum serviço fique
  temporariamente indisponível.
- A busca de memórias continua compatível com os dados existentes, sem exigir
  nova indexação das conversas.
