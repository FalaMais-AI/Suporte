# Versão 1.37.0

_Publicada em 20/07/2026._

## ✨ Novidades

- Na integração com a Digital Grid, agora você pode colar e salvar a chave de
  segurança enviada pelo provedor diretamente nas configurações do FalaMais.AI.
- Ao substituir uma chave já configurada, o sistema pede confirmação para
  reduzir o risco de interromper o recebimento de novos eventos por engano.
