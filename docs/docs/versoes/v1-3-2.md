# Versão 1.3.2

_Publicada em 27/05/2026._

## ✨ Novidades

- **Envio por link para integrações externas.** Integrações que só conseguem
  chamar uma URL agora podem disparar mensagens de confirmação pelo WhatsApp
  usando a API pública, incluindo botão de confirmação com link.

📖 **Saiba mais:** [Mensagens](../msg.md)

## 🔧 Melhorias

- **Mais compatibilidade para confirmações de agenda.** O envio por link aceita
  telefone, texto da mensagem, dados de confirmação e até três botões de link,
  facilitando integrações com sistemas de agenda que não enviam JSON por POST.

📖 **Saiba mais:** [Mensagens](../msg.md)
