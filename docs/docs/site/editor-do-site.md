# Editor do site

O editor do site permite alterar textos, imagens e posts do site institucional
sem mexer no código.

## Como acessar

1. Acesse `https://falamais.ai/admin`.
2. Clique para entrar com GitHub.
3. Autorize o acesso quando o GitHub solicitar.
4. Aguarde o painel carregar as páginas disponíveis para edição.

## O que pode ser editado

No painel, é possível editar páginas institucionais, textos de SEO, chamadas de
ação, perguntas frequentes, conteúdos de contato e posts do blog.

As alterações são salvas no repositório do projeto. Depois da publicação, o site
é atualizado pelo processo de deploy do marketing.

## Quem pode editar

A pessoa precisa ter acesso ao repositório do FalaMais.AI no GitHub. Para novos
editores, peça para um administrador adicionar o usuário GitHub como colaborador
com permissão de escrita.

## Boas práticas

- Revise título, descrição e links antes de publicar.
- Use imagens leves e com nomes claros.
- Não cole scripts, códigos externos ou estilos personalizados nos campos do
  editor.
- Para mudanças grandes, publique primeiro uma versão de rascunho e revise com
  o time.
