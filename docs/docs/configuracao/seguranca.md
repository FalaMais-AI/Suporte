# Segurança

Gerencie credenciais de acesso à plataforma.

## Email de acesso

Exibe o email principal utilizado para login.

Ação disponível:
Alterar email

O novo email será usado para:
- Login
- Recuperação de senha
- Comunicações da conta

## Senha

Permite alterar a senha da conta.

Recomendações:
- Utilize senha forte
- Atualize periodicamente
- Não compartilhe credenciais

Clique em Alterar senha para atualizar.
