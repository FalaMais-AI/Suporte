# ⚙️ Configurações

A seção Configurações permite gerenciar os parâmetros administrativos da conta, incluindo estrutura comercial, segmentação, personalização de dados e regras operacionais.

O menu lateral inclui:

- Conta
- Empresa
- Tags
- Campos personalizados
- Canais
- Usuários
- Pagamentos
- Notificações
- Integrações
- Aparência
- Segurança
- Mensagens rápidas
