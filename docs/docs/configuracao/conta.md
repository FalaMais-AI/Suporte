# Conta

A aba Conta permite gerenciar suas informações pessoais dentro da plataforma.

Aqui você pode atualizar sua foto de perfil, nome, telefone e fuso horário utilizado no sistema.

Essas informações impactam sua identificação interna e o funcionamento correto de registros baseados em horário.

![Tela de configurações da conta](img/image-6.png)

## Foto de Perfil

Permite personalizar sua imagem de usuário exibida na plataforma.

Funcionalidades:

- Alterar foto atual
- Upload de nova imagem

Formatos permitidos:

- PNG
- JPG
- GIF

Tamanho máximo:

- 20MB

Para alterar:

1. Clique em "Selecionar foto".
2. Escolha a imagem desejada.
3. Confirme o envio.

A nova imagem será associada ao seu perfil de acesso.

## Informações Pessoais

Permite atualizar seus dados básicos de identificação.

Campos disponíveis:

### Nome completo

Define como seu nome será exibido na plataforma, incluindo:

- Conversas
- Histórico interno
- Registros operacionais

### Telefone

Campo para cadastro do seu número de contato.

Formato recomendado:

(DD) 00000-0000

### Fuso horário

Define o fuso utilizado para:

- Registro de mensagens
- Relatórios
- Métricas de atendimento
- Histórico de atividades

Exemplo:

America/Sao_Paulo

:::warning Atenção:

Se o fuso horário estiver incorreto, os horários exibidos no sistema podem ficar inconsistentes.
:::

Após atualizar qualquer campo, clique em:

**Salvar alterações**

As informações serão atualizadas imediatamente no sistema.
