# Aparência

Permite personalizar o tema visual da interface.

![Tela de configuração de aparência](image-7.png)

Opções disponíveis:
- Claro
- Escuro
- Sistema (segue configuração do sistema operacional)

A alteração é aplicada imediatamente ao usuário atual.
