# Programa de Indicações

O **Programa de Indicações** recompensa quem indica o FalaMais.AI para novos
clientes.

## Como funciona

- Você indica o FalaMais.AI para outras empresas.
- Cada indicação é **registrada e acompanhada** na plataforma.
- Indicações convertidas geram **créditos** para a sua conta.

## Acompanhamento

A área de indicações mostra o **status** de cada indicação feita, permitindo
acompanhar quais já foram convertidas e quantos créditos foram gerados.

## Gestão pelo administrador

O administrador conta com uma área de **gestão de indicações**, onde é
possível acompanhar o programa e administrar os créditos concedidos.

:::tip
Os créditos do programa podem ser usados na sua assinatura — veja a página de
[Pagamentos](./configuracao/pagamentos.md).
:::
